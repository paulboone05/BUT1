# King of the Quizz : le jeu pour détrôner Lebron James

## Présentation de King of the Quizz 

Ce jeu est un jeu éducatif pour 2 joueurs qui proposent des questions à différentes difficultés aux élèves (facile, intermédiaire, difficile). 
Une question difficile rapporte 3 points, une question intermediaire rapporte 2 points et une question facile rapporte 1 point. 
Chacun leur tour les élèves auront un nombre de déplacement aléatoire attribué entre 6 et 17. 
Grâce à ce nombre de déplacement, ils pouront se déplacer sur le terrain de basketball pour s'approcher du panier et avoir des questions plus simples.
Le premier joueur à 10 points l'emporte !
Des captures d'écran illustrant le fonctionnement du jeu sont proposés dans le dossier shots.

## Utilisation de King of the Quizz

Afin d'utiliser le projet, il doit être suffisant de mettre le terminal en plein écran et de taper les commandes suivantes depuis le répertoire src :

ijava compile main.java //compile les fichiers présent dans src et crée les fichiers .class
ijava execute main //lance le jeu
